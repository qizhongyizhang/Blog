Python-基础

发表于 2021-07-08 更新于 2021-11-17 分类于 基础 阅读次数： Valine：

写在前面：本文主要记录Python基础用法

数据类型

Pyhton 没有数据类型关键字，在赋值时确定对应变量的数据类型，能够直接处理的数据类型有：整数、浮点数、字符串、布尔值、列表、元组、集合、字典

输入输出

**input()：接收输入，input([输入提示])**，保存为对应字符串

**print()：显示输出，print(打印内容 [, sep = ‘ ‘, end = ‘\n’])**，多个打印内容书写时用 , 隔开；打印时以 sep 隔开，默认为空格；以\n结束

  • 格式化输出：print(“XXX<占位符>”%(参数1,参数2))

控制语句

引用语句

  • import <库函数名>
  • from <库函数名> import <特定函数名>

循环语句

  • for遍历循环：for <循环变量> in <遍历结构>: …
  • 增强for循环：for … else … ，正常循环结束时执行else的语句内容
  • while无限循环：while <条件>: …
  • 循环控制：

break：直接跳出循环语言，执行循环后的语句
continue：退出本次循环过程，继续下一次循环

分支语句

  • 单分支：if <条件>: …
  • 双分支：if <条件>: …else <条件>: …
  • 多分支：if <条件>: …elif<条件>: … else <条件>: …

赋值语句

  • 多个变量同时赋值时，按照顺序赋值

异常控制语句

  • try: … except: … finally

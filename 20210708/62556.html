写在前面：本文主要记录Python的基础知识

Python解释器

CPython：官方版本的解释器：CPython，用C语言开发，是使用最广的Python解释器；用 >>> 作为提示符
IPython：是基于CPython之上的一个交互式解释器，在交互方式上有所增强，但是执行Python代码的功能和CPython是完全一样；用 In [序号]: 作为提示符
PyPy：采用JIT技术，对Python代码进行动态编译，显著提高Python代码的执行速度；执行结果可能会与 CPython 有所不同
Jython：是运行在Java平台上的Python解释器，可以直接把Python代码编译成Java字节码执行
IronPython：是运行在微软.Net平台上的Python解释器，可以直接把Python代码编译成.Net的字节码

保留关键字

保留关键字：33个，已经或者将要被赋予特定意义的单词，不能使用用这些保留字作为标识符给变量、函数、类、模板以及其他对象命名

查看关键字: print(keyword.kwlist)

控制语句

引用控制：import（导入文件、外部库、模块）、from
循环结构：while、for、break、continue
分支结构：if、elif、else
异常控制：try、except、finally、raise（抛出异常）、assert（设置断言，用来辅助查看，用于调试）

变量修饰符

class：定义类
def：定义函数
lambda：匿名函数
return：定义返回值
global：申明全局作用域
nonlocal：外部嵌套函数内的变量

逻辑判断

True：布尔是
False：布尔非
or：逻辑或
and：逻辑与
not：逻辑非
is：逻辑是
in：在某个范围内

其他

as
del
None
pass
with
yield：生成器返回申明

注：没有数据类型关键字，在赋值时确定对应变量的数据类型，支持数值类型有：整数类型（int）、浮点数类型（float）、复数类型（complex）、字符串（没有char保留字）、集合（set）、列表（list）、字典（dict，也叫映射）；随时命名、随时赋值（=）、随时使用

其他

命名规范

由字母、下划线、数字组成、汉字组成，首字符不能是数字，不推荐使用汉字
严格区分大小写，可以采用下划线命名、大驼峰、小驼峰等命名法；可以使用保留关键字的大写形式命名，但不推荐
常用全部大写的变量名表示常量

注释

  • 单行注释：#
  • 多行注释：’’’…’’’/“””…”””

输入输出

  • **input()：接收输入，input([输入提示])**，保存为对应字符串
str = input('Please entry:')  # 用 str 接收来自键盘的输入

num = eval(input('Please entry:'))  # 通过 eval() 将字符串行的数字转换为数值型
num = int(input('Please entry:'))  # 通过 int() 将字符串行的数字转换为整数型
num = float(input('Please entry:'))  # 通过 float() 将字符串行的数字转换为浮点型
  • **print()：显示输出，print(打印内容 [, sep = ‘ ‘, end = ‘\n’])**，多个打印内容书写时用 , 隔开；打印时以 sep 隔开，默认为空格；以\n结束
>>> print('hello', 'world')
hello world

>>> print(100 + 200)  # 可计算结果
300

>>> print('100 + 200 =', 100 + 200)  # 100 + 200 = 用 '' 包裹，表示字符串，而非表达式
100 + 200 = 300

格式化输出：

  1. 占位符：print(“XXX 占位符 “%(value1,value2))

    %s：字符串类型占位符
    %d：整数类型占位符（直接截取整数位）
    %f：浮点型占位符
    %x：16进制数占位符

    print('Age: %s, Gender: %s, growth rate: %d %%' % (25, True, 7));
    'Age: 25, Gender: True, growth rate: 7 %'
    输出内容中有 % 号时，需要用两个 %% 表示

  2. format：print(‘XXX {字符填充序号 : [填充字符] [对齐方式] [数据宽度] [千位分隔符] [保留小数位数] [数据类型]}’.format(value1,value2))

字符填充序号：format() 里面的数据索引，从零开始
填充字符：数据长度不够时的填充字符，默认用空格进行填充
对齐方式：数据对齐方式，^, <, > 分别是居中、左对齐、右对齐
数据宽度：数据的实际显示长度，用正整数表示，默认宽度为10，数据超过宽度则省略后宽度之后的数据
千位分隔符：数据级每隔 1000 的分隔标识符
保留小数位数：用 .+正整数表示，如 .4 表示保留 4位整数
数据类型：d 表示整数，f 表示浮点数
>>> print('{0:@<20,.4f}'.format(123456789.987654321))
123,456,789.9877@@@@

显示百分比：print('{0:-<30,.4%}'.format(123456789.987654321))
12,345,678,998.7654%----------
科学计数法：print('{0:-<30,.4e}'.format(12345646577886789.987654321))
1.2346e+16--------------------

二进制：'{:b}'.format(11)
十进制：'{:d}'.format(11)
八进制：'{:o}'.format(11)
普通十六进制：'{:x}'.format(11)
小写十六进制带b：'{:#x}'.format(11)
大写十六进制带B：'{:#X}'.format(11)
  1. f-string：**var定义（换行）print(f’XXX {var_name[:format格式]}’)**，使用 f 开头申明，用前面申明的对应变量替换相同位置的占位符
    >>> r = 2.5
    >>> s = 3.14 * r ** 2
    >>> print(f'The area of a circle with radius {r} is {s:.2f}.')
    The area of a circle with radius 2.5 is 19.62.

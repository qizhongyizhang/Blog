Anaconda-安装使用

发表于 2021-07-10 更新于 2021-09-15 分类于 安装使用 阅读次数： Valine：

安装

  1. 下载地址： 官网
  2. 选择好对应的位数和版本，双击下载好的exe文件，开始安装
  3. 选择All Users，下一步；
  4. 选择安装路径，下一步；
  5. 加入到环境变量（可选），下一步；
  6. 开始安装，时间较久；
  7. 完成测试：在cmd下运行conda –version

Jupyter

Jupyter：是 Anaconda 自带的一个工具，可以实时展示数据

使用

启动：命令行下输入：jupyter notebook，回车；即可在设置的浏览器上打开 jupyter

设置默认浏览器

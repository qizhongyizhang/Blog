安装第三方库

  • pip安装

修改pip镜像源下载地址

Windows：在用户主目录（C:/User/user_name）下新建pip文件夹，新建pip.ini文件：输入内容

[global]
index-url=https://pypi.doubanio.com/simple

Linux：在用户主目录（cd ~）下新建.pip文件夹，新建pip.conf文件：输入内容

[global]
index-url=https://pypi.doubanio.com/simple
将安装依赖项导出文件：pip freeze > requirements.txt
安装依赖项安装：pip install -r requirements.txt

Jupyter

  • 安装：pip install jupyter
  • 更新：pip install –upgrade pip
  • 启动：jupyter notebook

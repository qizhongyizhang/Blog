Python基础-函数

发表于 2021-07-09 更新于 2021-09-03 分类于 常见操作指令 阅读次数： Valine：

写在前面：本文主要记录Python的简单函数

简单函数

range(<起始位置>:<结束位置>:<步长>)（包含起始位置，不包含结束位置）

  • 下标有正负两种，正数：从左边开始数，以0开始；负数：从右边开始，从-1开始
  • 起始位置：可省略，省略表示从头开始
  • 结束位置：可省略，省略表示截取到最后
  • 步长：截取的刻度，正数从前往后截，负数从后往前截

格式化：’{<序号>:<填充字符><对齐方式><宽度><千位分隔符><保留小数位数><类型>}’.format()

数据运算

数值类型

  • 一元操作符：+、-、*、/、//（整数商）、%（求模或取余）、**（幂运算）
  • 二元操作符：+=、-=、*=、/=、//=、%=、**=
  • 返回绝对值：abs()
  • divmod（x，y）=（x//y，x%y）
  • pow（x，y）=x**y、pow（x，y，z）=x**y%z
  • round（x，d）：x四舍五入保留d位小数，省略d则保留整数
  • max（）、min（）

字符串类型：单一字符的有序组合

  • 读取（切片）：<字符串变量>[序号：序号]（不包括结束位置）
  • 操作符：连接（a+b）、复制（a * n：将a复制n次）、判断存在（b in/(not in) a：判断b是否在a中存在）
  • a.index(x)：返回第一次出现x的位置标号
  • a.count(x)：返回x在序列中出现的次数
  • len（x）：返回x字符串长度
  • str（x）：将x转换为字符串格式
  • chr（x）：返回Unicode编码对应的字符
  • ord（x）：返回字符对应的Unicode编码
  • hex（x）：返回x对应的十六进制数
  • oct（x）：返回x对应的八进制数
  • str.lower()：返回str小写形式的副本
  • str.upper()：返回str大写形式的副本
  • str.split(sep=’’)：返回根据sep字符分割的字符串列表
  • str.count()：返回子串出现次数
  • str.replace(old,new)：返回old被new替代的新字符串
  • str.center(width,fillchar)：字符串根据给定宽度居中显示，多余位置按fillchar填充
  • str.strip(char)：去掉字符串两端的指定char字符

集合类型：无顺序问题，输入不限，输出自动去重

  • 申明：a = set{}
  • a-b：在集合a却不在集合b中的元素
  • a&b：同时在集合a、b中的元素
  • a^b：集合a、b各自独有的元素
  • a|b：集合a、b共有的元素
  • len(a)：返回集合元素个数
  • a.add()：添加元素
  • a.remove()：移除元素
  • a.clear()：清空集合
  • x in/(not in) a：判断x是否在集合内

列表类型：

  • 申明：a = []
  • 读取（切片）：<字符串变量>[序号：序号]（不包括结束位置） / for <循环变量> in <列表变量>: <语句块>
  • len（x）：返回列表长度
  • max（x）、min（x）：返回列表最大最小值
  • list（x）：返回x的列表类型
  • list.append(x):在列表末尾加入元素
  • list.insert(i,x)：在列表第i位插入元素
  • list.clear()：清空列表
  • list.pop(i)：从列表中取出第i位元素并删除
  • list.remove(x)：删除列表中第一次出现的x
  • list.reverse()：将列表倒序排列
  • list.copy()：返回列表的副本
  • 删除：del <列表变量> [<索引起始>:<索引结束>]（不包含结束位置）

字典类型：由键值对表示，键必须唯一且不允许为空

  • 申明：d = {<键>:<值>}/dict()
  • 索引：<接受变量> = <字典名称>[<键>]
  • 遍历：for <变量名> in <字典名>:<语句块>
  • len（x）：返回字典键值对数量
  • max（x）、min（x）：返回键的最大最小值
  • d.keys()：返回字典的所有键
  • d.values()：返回字典所有的值
  • d.items()：返回字典所有的键值对
  • d.get(key,default)：若key存在，则返回对应值，否则返回默认值
  • d.pop(key,default)：若key存在，返回对应值并删除该键值对，否则返回默认值
  • d.popitem()：随机从字典中选取一个键值对，以元组形式输出，并删除该键值对
  • d.clear()：清空字典
  • 删除：del <字典名>[键]

数据类型转换：type（返回类型）、int（返回整数类型）、float（返回浮点数型）、str（返回字符串类型）

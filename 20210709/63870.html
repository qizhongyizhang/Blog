Python基础-函数

2021-07-09 常见操作指令 2.1k 字
Python

写在前面：本文主要记录Python的简单函数

函数介绍

自定义函数

自定义函数：定义一个函数要使用 def 语句，依次写出函数名、参数列表和冒号 : ，在缩进块中编写函数体，函数的返回值用 return 语句返回

def f_name([p_list]):
    xxx
    [return xxx]

def my_abs(x):   # 返回给定参数的绝对值
    if x >= 0:
        return x
    else:
        return -x

def f_name():
    pass     # 定义空函数，如果还没想好怎么写函数的代码，可以先放一个pass占位符，让代码能运行起来；缺少 pass，代码运行会有语法错误

函数类型

  • 无参无返回值
  • 无参有返回值
  • 有参无返回值
  • 有参有返回值

函数作为返回值：把函数作为结果值返回，在函数内部再定义一个函数，父层函数的返回值是子层函数

def lazy_sum(*args):
    def sum():
        ax = 0
        for n in args:
            ax = ax + n
        return ax
    return sum

f = lazy_sum(1, 3, 5, 7, 9)   # 调用 lazy_sum ，返回 f/sum 函数
f()  # 调用 f() 函数，显示输出 25

匿名函数：关键字 lambda 表示匿名函数；只能有一个表达式，不用写 return ，返回值就是该表达式的结果；不需要显式地定义函数，直接传入匿名函数更方便

f = lambda x: x * x

f(5)  # 输出： 25

函数参数

位置参数：按照参数的位顺序进行传递

def power(x):     # 计算 x^2 x的平方
    return x * x

num = power(5)    # 计算 5的平方

def power(x, n):  # 计算 x 的 n 次方，此时计算 x的平方 需要按照参数位置传入两个参数 
    s = 1
    while n > 0:
        n = n - 1
        s = s * x
    return s

num = power(5, 2)  # 计算 5的平方
num = power(5, 3)  # 计算 5的立方

关键字参数：用于限制关键字参数的名字，命名关键字参数需要一个特殊分隔符 * 号， * 号后面的参数被视为命名关键字参数，只能通过关键字的方式传递参数，即以 p_name = value 形式进行参数传递

def person(name, age, *, city, job):
    print('name:', name, 'age:', age, 'city:', city, 'job:', job)

person('Jack', 24, city='Beijing', job='Engineer')
name: Jack age: 24 city: Beijing, job: Engineer

默认参数：对于某一个常用的参数取值设置成默认值，如上文：经常计算 x的平方，可以将计算平方设置成默认值；默认参数必须指向不变对象

def power(x, n=2):   # 计算 x 的 n 次方，默认计算 x的平方
    s = 1
    while n > 0:
        n = n - 1
        s = s * x
    return s

num = power(5)     # 计算 5的平方
num = power(5, 3)  # 计算 5的立方

def add_end(L=[]):  # 多次使用默认参数调用时，数据会出错
    L.append('END')
    return L
Python 函数在定义的时候，默认参数L的值就被计算出来了，即[]；但默认参数L也是一个变量，指向对象[]；每次调用该函数，函数体改变了L的内容，则下次调用时，默认参数的内容就变为['END']，不再是函数定义时的[]

可变位置参数：传入的参数个数可变，定义可变参数在参数名前加一个 * 号进行申明，可变参数在函数调用时自动组装为一个 tuple

def calc(numbers):    # 计算 1~n 的平方和
    sum = 0
    for n in numbers:
        sum = sum + n * n
    return sum
result = calc([1, 2, 3, 4, 5])    # 调用时需要先组装出一个 list 或 tuple 进行传递
result = calc((1, 2, 3, 4, 5]))

def calc(*numbers):    # 参数 numbers 接收到的是一个 tuple
    sum = 0
    for n in numbers:
        sum = sum + n * n
    return sum
result = calc(1, 2, 3, 4, 5)  # 可以直接传递任意个数据，包括 0 个
result = calc(*list_a)  # 对于前文的 list 或 tuple 可以直接在变量名前添加 * 号把 list 或 tuple 的元素变成可变参数进行传递

可变关键字参数：允许传入任意个含参数名的参数，定义可变参数在参数名前加两个 * 号进行申明，关键字参数在函数内部自动组装为一个 dict

def person(name, age, **kw):
    print('name:', name, 'age:', age, 'other:', kw)

extra = {'city': 'Beijing', 'job': 'Engineer'}
person('Jack', 24, **extra)
输出 ： name: Jack age: 24 other: {'city': 'Beijing', 'job': 'Engineer'}

参数组合：函数的参数可以是上面5种参数的任意组合；参数定义的顺序必须是：必选参数、默认参数、可变位置参数、关键字参数和可变关键字参数

def func_1(a, b, c = 0, *args, d, **kw):     # 如果函数定义中有可变位置参数，后面跟着的关键字参数就不再需要特殊分隔符 *
    print('a = ', a, 'b = ', b, 'c = ', c, args, 'd = ', d, kw)

func_1(1, 2, 3, 4, 5, d = 6, e = 7, f = 8)
输出：1 2 3 (4, 5) 6 {'e': 7, 'f': 8}

查看函数

查看函数信息：help(f_name)

调用函数

调用函数：通过函数名进行调用，给定正确的参数，包括参数个数个数据类型

num = abs(-12.5)  # 计算传入参数 -12.5 的绝对值大小
num = abs(12, 1)  # 会报 TypeError 的错误，提示传入的参数数量不对
num = abs('abs')  # 会报 TypeError 的错误，提示 str 是错误的参数类型

函数名其实就是指向一个函数对象的引用，完全可以把函数名赋给一个变量，相当于给这个函数起了一个“别名”

my_abs = abs
num = my_abs(-12.5)  # 计算传入参数 -12.5 的绝对值大小

简单函数

内置函数

**isinstance(value, datatype)**：数据类型检查，判断给定的数据是否是给定的数据类型，返回对应布尔值

num = 12
b = isinstance(num, string)  # 返回 False
b = isinstance(num, (int, float))  # 返回 True

range(<起始位置>:<结束位置>:<步长>)（包含起始位置，不包含结束位置）

  • 下标有正负两种，正数：从左边开始数，以0开始；负数：从右边开始，从-1开始
  • 起始位置：可省略，省略表示从头开始
  • 结束位置：可省略，省略表示截取到最后
  • 步长：截取的刻度，正数从前往后截，负数从后往前截

**map(f_name, Iterable)**：将传入的函数依次作用到序列的每个元素，并把结果作为新的 Iterator 返回；第一个参数是函数对象本身，Iterable 惰性序列

def f(x):
    return x * x

r = map(f, [1, 2, 3, 4, 5, 6, 7, 8, 9])
list(r)  # 显示输出 [1, 4, 9, 16, 25, 36, 49, 64, 81]

**reduce(f_name, values)**：把一个元素的函数计算结果继续和序列的下一个元素做函数计算

from functools import reduce
def fn(x, y):
    return x * 10 + y

reduce(fn, [1, 3, 5, 7, 9])   # 显示输出 13579

**filter(f_name, values)**：把传入的函数依次作用于每个元素，然后根据返回的布尔值决定是否保留该元素

def is_odd(n):
    return n % 2 == 1

list(filter(is_odd, [1, 2, 4, 5, 6, 9, 10, 15]))  # 显示输出 [1, 5, 9, 15]

**sort(values，key)**：按照 key 函数进行排序，是否以 reverse 方式进行呼出

sorted([36, 5, -12, 9, -21], key = abs, reverse = False)  # 显示输出 [5, 9, -12, -21, 36]，按照绝对值大小进行排序，顺序显示

递归函数

递归函数：在函数内部调用其它函数，如果在一个函数在内部调用自身本身，则被称为递归函数，使用递归函数需要注意防止栈溢出，可以通过尾递归进行优化；理论上，所有的递归函数都可以写成循环的方式，但循环的逻辑不如递归清晰
尾递归：是指在函数返回时，调用自身本身，并且，return语句不能包含表达式

def fact(n):       # 计算 n 的阶乘
    if n == 0:
        return 1
    else:
        return fact(n-1) * n
def fact(n):
    return fact_iter(n, 1)

def fact_iter(num, product):
    if num == 0:
        return product
    return fact_iter(num - 1, num * product)

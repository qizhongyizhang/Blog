Python基础-数据类型

发表于 2021-11-18 分类于 基础 阅读次数： Valine：

写在前面：本文主要记录Python的数据类型

数据类型

Pyhton 没有数据类型关键字，能够直接处理的数据类型有：整数、浮点数、字符串、布尔值、列表、元组、集合、字典
动态语言：变量本身数据类型不固定的语言，在赋值时确定对应变量的数据类型；如：Python
静态语言：在定义变量时必须指定变量类型，如果赋值的时候类型不匹配，就会报错；如：Java

基本数据类型

整数

整数：可以处理任意大小的整数，包括负整数；表示方法和数学上的写法一模一样，Python 允许在数字中间以 _ 分隔数值较大的数，方便阅读

浮点数

浮点数：指按照科学记数法表示时，浮点数的小数点位置可变，如：1.23x10^9 = 12.3x10^8 ，在 Python 里面写作 1.23e9

整数和浮点数在计算机内部存储的方式不同：整数运算永远是精确的，普通除法结果是浮点数，地板除 // 结果都是整数，浮点数运算则可能会有四舍五入的误差

数值处理函数

  • 一元操作符：+、-、*、/、//（整数商）、%（求模或取余）、**（幂运算）
  • 二元操作符：+=、-=、*=、/=、//=、%=、**=
  • 返回绝对值：abs()
  • divmod(x，y) = (x//y，x%y)
  • pow(x，y) = x**y、pow(x，y，z) = x**y%z
  • round(x，d)：x四舍五入保留d位小数，省略d则保留整数
  • max()、min()

字符串

字符串：是以单引号 ‘ 或双引号 “ 括起来的任意文本，如果文本内包含 ‘ 或 “ ，可以通过 \ 进行转义，也可以 ‘ 和 “ 混用

多行字符串：打印多行字符串时，可以通过 ‘’’ 或 “”” 括起来

字符串处理函数

  • 读取（切片）：str[start_num: end_num]，不包括结束位置
  • 操作符：连接（a+b）、复制（a * n：将a复制n次）、判断存在（b in/(not in) a：判断b是否在a中存在）
  • str.index(x)：返回第一次出现x的位置标号
  • str.count(x)：返回x在序列中出现的次数
  • len(str)：返回x字符串长度
  • str(str)：将x转换为字符串格式
  • chr(str)：返回Unicode编码对应的字符
  • ord(str)：返回字符对应的Unicode编码
  • hex(str)：返回x对应的十六进制数
  • oct(str)：返回x对应的八进制数
  • str.lower()：返回str小写形式的副本
  • str.upper()：返回str大写形式的副本
  • str.split(sep=’’)：返回根据sep字符分割的字符串列表
  • str.count()：返回子串出现次数
  • str.replace(old,new)：返回old被new替代的新字符串
  • str.center(width,fillchar)：字符串根据给定宽度居中显示，多余位置按fillchar填充
  • str.strip(char)：去掉字符串两端的指定char字符

布尔值

布尔值：仅代表是和否，用 True 、False 代替，注意首字母大写，可结合逻辑运算符 and、or、not 进行运算

复杂数据类型

列表

列表类型：申明：a = []

列表处理函数

  • 读取（切片）：<字符串变量>[序号：序号]（不包括结束位置） / for <循环变量> in <列表变量>: <语句块>
  • len(x)：返回列表长度
  • max(x)、min(x)：返回列表最大最小值
  • list(x)：返回x的列表类型
  • list.append(x):在列表末尾加入元素
  • list.insert(i,x)：在列表第i位插入元素
  • list.clear()：清空列表
  • list.pop(i)：从列表中取出第i位元素并删除
  • list.remove(x)：删除列表中第一次出现的x
  • list.reverse()：将列表倒序排列
  • list.copy()：返回列表的副本
  • 删除：del <列表变量> [<索引起始>:<索引结束>]（不包含结束位置）

字典

字典类型：由键值对表示，键必须唯一且不允许为空，申明：d = {<键>:<值>}/dict()

字典处理函数

  • 索引：<接受变量> = <字典名称>[<键>]
  • 遍历：for <变量名> in <字典名>:<语句块>
  • len(x)：返回字典键值对数量
  • max(x)、min(x)：返回键的最大最小值
  • d.keys()：返回字典的所有键
  • d.values()：返回字典所有的值
  • d.items()：返回字典所有的键值对
  • d.get(key,default)：若key存在，则返回对应值，否则返回默认值
  • d.pop(key,default)：若key存在，返回对应值并删除该键值对，否则返回默认值
  • d.popitem()：随机从字典中选取一个键值对，以元组形式输出，并删除该键值对
  • d.clear()：清空字典
  • 删除：del d_nam3[key]

集合

集合：无顺序问题，输入不限，输出自动去重；申明：a = set{}

集合处理函数

  • a-b：在集合a却不在集合b中的元素
  • a&b：同时在集合a、b中的元素
  • a^b：集合a、b各自独有的元素
  • a|b：集合a、b共有的元素
  • len(a)：返回集合元素个数
  • a.add()：添加元素
  • a.remove()：移除元素
  • a.clear()：清空集合
  • x in/(not in) a：判断x是否在集合内

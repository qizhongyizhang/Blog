使用

基础指令

  • 初始化：hexo init [folder]
    如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站
  • 新建：hexo new，简写：hexo n
    • 新建文件夹：hexo new page page_name，如果名字里面有空格，需要用英文双引号””包裹，下面也一样，默认在source/目录下新建，可通过path参数进行修改，例：hexo new page –path xxx/xxx，同时也会生成一个index.md的文件
    • 新建文件：hexo new post_name，默认是在当前目录下生成
      More info: Writing
  • 生成：hexo generate，简写：hexo g
    More info: Generating
  • 发布：hexo server ，简写：hexo s
    More info: Server
  • 部署：hexo deploy，简写：hexo d
    More info: Deployment
  • 清除：hexo clean，简写：hexo c

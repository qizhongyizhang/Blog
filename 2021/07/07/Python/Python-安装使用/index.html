安装

下载地址： 官网

安装第三方库

  • pip安装

修改pip镜像源下载地址

Windows：在用户主目录（C:/User/user_name）下新建pip文件夹，新建pip.ini文件：输入内容

[global]
index-url=https://pypi.doubanio.com/simple

Linux：在用户主目录（cd ~）下新建.pip文件夹，新建pip.conf文件：输入内容

[global]
index-url=https://pypi.doubanio.com/simple
将安装依赖项导出文件：pip freeze > requirements.txt
安装依赖项安装：pip install -r requirements.txt

通识

保留关键字

保留关键字：33个，已经或者将要被赋予特定意义的单词，不能使用用这些保留字作为标识符给变量、函数、类、模板以及其他对象命名

控制语句

循环结构：while、for、break、continue
分支结构：if、elif、else
异常控制：try、except、finally

变量修饰符

class：定义类
def：定义函数
return：定义返回值
global：申明全局作用域

逻辑判断

True：布尔是
False：布尔非
or：逻辑或
and：逻辑与
not：逻辑非
is：逻辑是
in：在某个范围内

其他

as
assert
del
from
import：导入文件、外部库
lambda
None
nonlocal
pass
raise
with
yield

注：没有数据类型关键字，在赋值时确定对应变量的数据类型，支持数值类型有：整数类型（int）、浮点数类型（float）、复数类型（complex）、字符串（没有char保留字）、集合（set）、列表（list）、字典（dict，也叫映射）；随时命名、随时赋值（=）、随时使用

命名规范

由字母、下划线、数字组成、汉字组成，首字符不能是数字，不推荐使用汉字
严格区分大小写，可以采用下划线命名、大驼峰、小驼峰等命名法；可以使用保留关键字的大写形式命名，但不推荐

注释

  • 单行注释：#
  • 多行注释：’’’…’’’/“””…”””

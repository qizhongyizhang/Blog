Hive基础-函数

写在前面：本文主要记录Hive的基础知识-函数

内置函数

查看内置函数：show functions;
查看内置函数的用法：desc function [extended] fun_name;使用 extended 会更加详细，带有示例

字符串函数

concat：拼接字符串

数据类型函数

cast：类型转换

get_json_object：解析 json 型数据

窗口函数

function_name([ col_name_list ]) over ( [partition by (col_name_list)] [ordr by (col_name_list)] [rows between … and …])

function_name : 计算类型
partition by : 分组字段
order by : 排序字段
rows between … and … : 计算的行数

rows between unbounded preceding and current row：包括本行和之前所有的行
rows between current row and current unbounded follow：包括本行和之后所有的行
rows between 3 preceding and current row：包括本行和前3行
rows between 3 preceding and 1 following：包括本行前3行和本行之后1行（共5行）

累计窗口函数

  • sum(…) over( [partition by ()] [ordr by ()] [rows between … and …] )
  • avg(…) over( [partition by ()] [ordr by ()] [rows between … and …] )
  • max(…) over( [partition by ()] [ordr by ()] [rows between … and …] )
  • min(…) over( [partition by ()] [ordr by ()] [rows between … and …] )

分组排序窗口函数

  • row_number() over( [partition by ()] [ordr by ()]) ：1、2、3
  • rank() over( [partition by ()] [ordr by ()])：1、1、3
  • dense_rank() over( [partition by ()] [ordr by ()])：1、1、2
  • ntile(n) over( [partition by ()] [ordr by ()]) : n 切片的个数

偏移分析窗口函数

  • lag(col_name_list, offset, defval) over( [partition by ()] [ordr by ()]) : 之前 offset 行的数据
  • lead(col_name_list, offset, defval) over( [partition by ()] [ordr by ()]) : 之后 offset 行的数据

offset : 偏移量

defval : 默认值，没有值时的默认值

其他

parse_url：解析 url 地址

explode：将 map 集合中的每一个键值对或数组中的每一个元素都单独生成一行

自定义函数

**UDF(user defined function)**：一进一出
继承 org.apache.hadoop.ql.execc.UDF 类，实现 evalute 函数，支持重载，必须拥有返回类型(返回值可以为 null ，返回类型不能为 viod )，推荐使用 Hive 数据类型

**UDAF(user defined aggregation function)**：多进一出
**UDTF(user defined generating function)**：一进多出

计算机-通识

2021-04-10 通识 2.3k 字
计算机

写在前面：本文主要记录计算机常识、历史知识等

简单介绍

发展过程

  • 算盘
  • 步进计算器：戈特弗里德·莱布尼茨（德国）建造于1694年，可以进行四则运算
  • 差分机：查尔斯·巴贝奇1823年开始制造，未完成；1991年完成
  • 分析机：查尔斯·巴贝奇（通用计算器、不止运算，有内存）
  • 打孔卡片制表机：赫尔曼·何乐礼，10倍手工；成立了制表机器公司，1924年于其他机械制造商合并为IBM（国际商用机器公司）

计算机之父

通用计算机之父——巴贝奇：（1792—1871、英国）设计了第一个可编程计算机，完成计算机构想（冯·诺伊曼）
计算机科学之父——图灵：（1912-1954、人工智能之父、英国伦敦、同性恋）破解德国的著名密码系统Enigma（英格码）、制出了图灵机、图灵测试
电子计算机之父——约翰·阿坦那索夫：（1903-1995、保加利亚移民）制造ABC计算机、使用二进制
现代计算机之父——冯·诺伊曼：（1903-1957、美籍匈牙利、博弈论之父）冯·诺伊曼模型、博弈论

计算机组成

数据存储

介质

发展过程：打孔纸卡/带 —> 延迟线存储器（1944、压力波，每时刻只能读一位，顺序存储器/循环存储器） —> 磁芯存储器（1953麻省理工学院Whirlwind 1） —> 磁带（访问速度慢，顺序读取） —> 磁鼓存储器 —> 软盘（1970-1990，磁盘是软的） —> 硬盘 —> 光盘（光反射） —> U盘

  • 磁盘：永久性存储，非易失性存储器，速度慢
    磁盘存储是一种稳定的永久存储数据库方法；磁盘盘片的逻辑结构主要分为磁道、扇区和柱面，一个盘面分成多个磁道(同心圆环，300~1024个)，一个词到分成多个扇区，扇区时磁盘最小的存储单位，512个字节
    读取数据：磁头根据磁盘地址找到对应的磁道，等磁盘旋转到对应的扇区才能获取到数据，一般有十几毫秒的时延
    写入数据：有两种方式：随机写(数据文件、增删很频繁)和顺序写(日志文件)，随机写的速度低于顺序写
  • 存储区域网络：SAN，Storage Area Networks，磁盘阵列组成存储区域网络
  • 内存：系统启动时，内存数据库（IMDB，In-Memory Database）从永久存储中将数据加载到内存中，所有数据处理都在内存中完成；非永久存储，断电清空，易失性存储器，速度快
  • 列压缩方案：列式数据库（Columnar-based Database）用于处理大量重复的数据集存储
  • 闪存：Flash Memory

方式

数据存储方式：数据以二进制数进行存储

  • 游程编码：无损，消除冗余
  • 字典编码：无损，霍夫曼编码（霍夫曼树）
  • 感知编码：有损，删掉人类无法感知的数据（音频、图片、视频（时间冗余，只存改变的部分））

人机交互

命令行

  1. 打印机：打印到纸上
  2. 打孔纸卡、磁带
  3. 电传打字机：QWERTY键盘（肖尔斯）
  4. cmd/Terminal（Mac）

图形界面

  1. 阴极射线管：发射电子到磷光涂层屏幕发光，绘制图形/像素（矢量扫面/光栅扫描）
  2. 用符号代替像素，用字符生成器（第一代显卡、ASCLL码，只读存储器ROM存储点阵图案）从内存中读取像素转换成字符，利用屏幕缓冲区投射到屏幕上
  3. 扩展字符：CRT的矢量模式（划线表示所有）
  4. 交互图形界面：Sketchpad（1962，辅助计算机设计、CAD前身）光笔（确定位置）
  5. 显示器：使用像素（1960年代末），位图显示
  6. 现代图形界面：（道格拉斯·恩格尔巴特：二战菲律宾雷达操作员，1955在UCB取得博士学位，1964年创造出鼠标）GUI计算机事件驱动编程（施乐奥托，1973）桌面隐喻（WIMP界面面：窗口、图标、菜单、指针）
  7. 3D图形：3D投影（3D到2D、框架渲染：正交投影/透视投影；不规则用三角形（多边形）填充图形：扫描线渲染算法、抗锯齿，画家算法（排序）/深度缓冲（不排序，记录像素与摄像机的距离）；背面剔除，明暗处理（表面法线、高洛德着色、冯氏着色），纹理化，GPU：图像处理单元）

操作系统

发展过程

  • 开始于1950年代，计算机运行速度更快，提供API（设备驱动程序）协调外部设备
  • 批处理：加强程序加载方式，一次给多个程序，自动运行下一个
  • 多任务处理：添加标志位，利用外部设备的响应时间读取运行下一道程序
  • 分时操作系统：每个用户只能使用一部分处理器、内存（Multics多任务信息与计算系统，1969，直接考虑安全）
  • Unix，70、80年代最流行的操作系统，核心功能：内存管理、多任务输入/输出处理（内核），通过程序和运行库运行
  • 磁盘操作系统（MS-DOS）：1981发布，160kB，简单，适应个人电脑，微软（没有内存保护——“蓝屏”）
  • 现代操作系统：Mac OS X、Windows10、Linux、iOS、Android

计算机网络

网络安全类

  • 后门：指计算机系统或应用程序的忽略隐藏入口，允许未经授权用户绕过密码等限制获取访问权限
  • 机器人或僵尸：是指已被恶意黑客使用特洛伊木马、病毒、网络钓鱼或下载受感染文件接管的工作站，被远程控制执行恶意任务
  • Cookie：是网站在计算机硬盘上暗访的小型数据文件，用于识别老用户并分析其偏好
  • 防火墙：是过滤网络流量的软硬件，用于保护单个计算机或整个网络免受未经授权的访问和免遭企图对系统的攻击
  • 周界：指组织环境与外部系统之间的边界
  • DMZ：De-militarized Zone，非军事区的简称，指系统边缘或外围区域，内部系统 —> DMZ —> 互联网，三者两两之间都有防火墙
  • 键盘记录器：对键盘键入的所有击键进行记录，返回给到攻击者，以捕获用户名、密码、网址等
  • 渗透测试：通过白帽黑客的外部侵入，验证系统安全性所做的测试
  • VPN：虚拟专用网络，使用不安全的互联网创建进入组织环境的安全路径或高度加密的隧道，对数据进行强加密，允许内部网络通信，需要经过多重身份验证连接到组织环境外围
  • 特洛伊木马：通过伪装或嵌入合法软件进入计算机系统的恶意程序

定理定律

摩尔定律：摩尔定律是由英特尔（Intel）创始人之一戈登·摩尔提出：当价格不变时，集成电路上可容纳的元器件的数目，约每隔18-24个月便会增加一倍，性能也将提升一倍

趣味统计

计算机之最

  • 最早计算机器算盘：公元前2500年在美索不达米亚发明（之后有星盘、计算尺、时钟）
  • 最早使用计算机：1613年，代指计算的人（职业），之后指机器
  • 第一个可编程的计算机：巨人1号（使用真空管，由汤米·弗洛斯完成于1943年12月）
  • 第一个可编程的通用计算机：ENIAC（1946年在宾夕法尼亚大学由约翰·莫奇利和普雷斯伯·埃克特完成）
  • 最大的计算机：哈佛马克一号（之一），IBM在1944年建造给二战同盟国，曼哈顿计划（速度慢、机器磨损、吸引昆虫）
  • 最早编程：雅卡尔织布机（1801、雅卡尔）——插线板编程——面板编程——语言编程
  • 第一个计算机程序员：阿达·洛夫莱斯（1815-1852），拜伦之女，写了第一个算法（分析记笔记）
  • 第一个完整的图形程序：Sketchpad
  • 第一台取得商业成功的计算机：Univac 1（40多台，型号）
  • 第一台磁盘计算机：RAMAC 305（IBM，5M，寻道时间1/6秒）
  • 第一台冯·诺伊曼架构储存程序计算机：宝宝（1948、曼彻斯特大学）
  • 第一台超级计算机：Atlas（50年代后期开始研究，1962年完成Atlas Supervisor，英国曼彻斯特大学）
  • 第一个单芯片CPU：英特尔4004（1971、4位、46条指令）英特尔74181（1970、封装在单个芯片内）
  • 第一个RAM集成电路：1972，每比特1美分

Sublime-安装使用

2021-04-09 安装使用 409 字
Tools

写在前面：本文主要记录Sublime Text 3的使用

安装

基础设置

  • 设置字体：依次点击首选项(Prefereences) -> 设置-用户(Setting Syntax Specific)，进入编辑弹窗，修改”font_size”: N，N为自己设定的大小，注意冒号之后有空格
  • 修改快捷键：依次点击首选项(Prefereences) -> 按键绑定-默认

快捷键Shortcuts

文件操作

  • 新建文件：Ctrl + N
  • 文件切换：Ctrl + Tab
  • 文件保存：Ctrl + S

文本操作

  • 整体后移：Tab
  • 整体后移：Shift + Tab

键盘输入

  • 文字剪切/复制/粘贴：Ctrl + X/C/V
  • 撤销操作：Ctrl + Z
  • 合并两行：Ctrl + J（光标位于上一行任意位置）
  • 文本缩进：Ctrl + [/]（左/右移Tab，光标位于本行任意位置）
  • 选择当前行：Ctrl + L（光标位于本行任意位置）
  • 块选择：Shift + 鼠标右键
  • 下方开辟新行：Ctrl + Enter（光标位于本行任意位置）
  • 上方开辟新行：Ctrl + Shift + Enter（光标位于本行任意位置）

光标定位

  • 光标左/右移：左/右箭头
  • 光标左/右移：Alt + 左/右箭头（移动一个单词）
  • 移动至本行行首：home
  • 移动至本行末尾：end

辅助工具

  • 打开命令面板：Ctrl + Shift + p；

常见问题

编码

问题1：设置GBK编码

原因：Sublime text默认不支持GBK编码
解决：下载插件支持
步骤：

  1. 打开控制面板：Ctrl + Shift + p
  2. 输入安装指令：Package Control：Install Package
  3. 选择安装插件：ConvertToUTF8，出现如下界面，则安装成功
  4. 重新打开或者设置编码：依次点击File -> Set File Encoding to/Reload with Encoding -> GBK，如下图

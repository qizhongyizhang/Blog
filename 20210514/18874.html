MySQL基础-函数

2021-05-14 常见操作指令 2.7k 字
数据库 MySQL

写在前面：本文主要记录MySQL函数的相关知识

聚合函数

  • 统计数目：COUNT(col_name)，确定表中行的数目或符合特定条件的行的数目，
COUNT(col_name)：统计指定列中不为null值的数据条数
COUNT(*)：对表中行的数目进行计数，包括空值NULL
COUNT(1)：返回所有的记录
  • 统计计算
MIN(col_name)     -- 返回指定列的最大值，只能计算一列，多列使用多个
MAX(col_name)     -- 返回指定列的最小值，只能计算一列，多列使用多个
SUM(col_name)     -- 返回指定列的总和，只能计算一列，多列使用多个
AVG(col_name)     -- 返回指定列的均值，只能计算一列，多列使用多个
  • 组内连接：GROUP_CONCAT(expr)

聚合函数通常与group by一起使用，用于统计组内的信息；单独使用，则将所有行看成一组，不统计null值

数学函数

Abs(X)           -- 绝对值，例：abs(-10.9) = 10
Ceil(X)          -- 向上取整，例：ceil(10.1) = 11
Cos()            -- 获取一个角度的余弦值
Exp()            -- 返回一个数的指数值
Floor(X)         -- 向下取整，例：floor (10.1) = 10
Format(X,D)      -- 格式化千分位数值，例：format(1234567.456, 2) = 1,234,567.46
Mod(M,N)         -- 求余 M%N / M MOD N，例：10%3=1
Pow(M,N)         -- 乘方 M^N
Rand(X)          -- 随机数
Pi()             -- 获得圆周率
Round(X)         -- 四舍五入去整
Sin()            -- 返回一个角度的正弦值
Sqrt(X)          -- 算术平方根
Tan()            -- 返回一个角度的正切值
TRUNCATE(X,D)    -- 截取D位小数

字符串函数

获取信息

  • **CHARSE(str)**：返回字串字符集
  • **CHAR_LENGTH(str)/LENGTH(str)**：获取字符串str长度
CHAR_LENGTH(str)：（汉字、数字、字母都是一个字符），例：
select CHAR_LENGTH('10,A,B,张');  -- 8
select CHAR_LENGTH('10,A,B');  -- 6
select CHAR_LENGTH(100);   -- 3

LENGTH(str)：获取string长度（字符数，一个汉字算三个字符，字母算一个字符，几位数字算几个字符），例：
select LENGTH('10,A,B,张');  -- 10
select LENGTH('10,A,B');  -- 6
select LENGTH(100);   -- 3
  • **INSTR(string, substring)**：从头寻找 string 中第一个 substring 出现的位置
  • **LOCATE(substring, string [,start_position])**：从 string 的 start_position 位置寻找第一个 substring 出现的位置

position：指定子串的起始位置，position可以是正整数、负整数和零；position为正整数，按照字符顺序编号从指定位置截取到字符串末尾；position为负整数，按照字符逆序编号从指定位置截取到字符串末尾；position为零，返回一个空字符串

  • **Soundex(string)**：返回字符串的Soundex值，即文本串转换为描述其语音表示的字母数字模式的算法值
  • **STRCMP(string1, string2)**：比较两字串大小：string1大，返回1；string2小，返回-1；二者一样大，返回0

拼接截取

  • **CONCAT(str1,str2[,… ])**：拼接字串，将str1、str2…顺序拼接起来，可以列名将字段值进行顺序拼接
CONCAT(id,"@@",name)，将id、@@和name里面的数据顺序拼接起来
  • **GROUP_CONCAT( [DISTINCT] col_name [Order BY col_name_list ASC/DESC] [Separator ‘分隔符’] )**：将col_name字段的数据按照col_name_list进行排序，以分隔符进行区分，默认分隔符为逗号
select group_concat(emp_no) as employees from dept_emp group by dept_no;将dept_emp表按照dept_no进行分组，将同一个dept_no的emp_no字段里的数据拼接起来，以逗号分隔，将所得的字段重命名为employees
  • 长度填充
LPAD(string, length, substr)：在 string 左边重复用 substr 填充直到字符串长度为 length
RPAD(string, length, substr)：在 string 右边重复用 substr 填充直到字符串长度为 length
  • 截取字符串
LEFT(string, length)：从 string 中的左边开始起取 length个字符
RIGHT(string, length)：从 string 中的右边开始起取 length个字符
SUBSTRING(str, position [,length])：从 str 的 position 开始，向右取 length 个字符，省略长度则全取出来

字符串修改

  • **REPLACE(str, search_str, replace_str)**：在str中用replace_str替换search_str
> select replace("10,A,B",",","");将字符串10,A,B里面的,替换为空——删除10,A,B里面的,
  • 去除空格
- LTRIM(string)：去除前端的空格
- RTRIM(string)：去除后端的空格
- TRIM(string)：去掉左右两边的空格
  • 大小写切换
LCASE('string')：转换为小写
Lower('string')：转换为小写
Upper('string')：转换为大写
UCASE('string')：转换为大写
  • **REPEAT(string, n)**：将 string 重复n次输出

时间日期函数

时间日期：推荐使用4位年进行存储，支持2位年，00-69为2000-2069，70-99为1970-1999

时间获取

  • 获取当前日期时间：NOW()/current_timestamp()，获取的是当前语句执行的时间，转换后的时间
  • 获取当前日期时间：sysdate()，获取的是sysdate()函数执行的时间，转换后的时间
  • 获得unix时间戳：Unix_timestamp()，时间戳
  • 从时间戳获得时间：From_unixtime()，将时间戳转换为常用时间格式YYYY-MM-DD hh:mm:ss
  • 获取当前日期：CURDATE()/Current_date()，使用：SELECT CURDATE(); 默认返回格式为YYYY-MM-DD
  • 获取当前时间：CURTIME()/current_time()，使用：SELECT CURTIME(); 默认返回格式为hh:mm:ss
  • 获取年份部分：Year(‘YYYY-MM-DD hh:mm:ss’)
  • 获取月份部分：Month(‘YYYY-MM-DD hh:mm:ss’)
  • 获取天部分：Day(‘YYYY-MM-DD hh:mm:ss’)
  • 获取日期部分：Date(‘YYYY-MM-DD hh:mm:ss’)
  • 获取时间部分：Time(‘YYYY-MM-DD hh:mm:ss’)
  • 获取小时部分：Hour(‘YYYY-MM-DD hh:mm:ss’)
  • 获取分钟部分：Minute(‘YYYY-MM-DD hh:mm:ss’)
  • 获取秒钟部分：Second(‘YYYY-MM-DD hh:mm:ss’)
  • 返回日期对应的星期几：DayOfWeek(‘YYYY-MM-DD hh:mm:ss’)
  • 返回日期/时间的某一部分：EXTRACT(unit FROM date), 使用：SELECT XTRACT(unit FROM date);

时间计算

  • **TIMESTAMPDIFF(unit,datetime1,datetime2)**：计算 datetime2 - datetime1在unit单位下的时间间隔
  • **DATEDIFF(datetime1,datetime2)**：计算 datetime1 - datetime2 相隔的天数
  • **DATE_ADD(datetime, INTERVAL number unit)**：计算 datetime 加上 number个unit单位的时间间隔
  • **DATE_SUB(datetime, INTERVAL number unit)**：计算 datetime 减去 number个unit单位的时间间隔
  • **to_days(‘YYYY-MM-DD hh:mm:ss’)**：返回从0000年（公元1年）到指定日期的总天数

时间操作

  • **sleep()**：沉睡指定时间，默认单位是秒
  • 日期格式化：date_format(< data(‘YYYY-MM-DD hh:mm:ss’) >,< format(‘%D %y %a %d %m %b %j’) >)，将 data 按照 format 的格式输出

流程控制函数

分支语句

  • IF语句
IF search_condition THEN 
statement_list
[ELSEIF search_condition THEN statement_list] 
...
[ELSE statement_list]END IF;
  • CASE语句
CASE case_value
WHEN when_value THEN statement_list
[WHEN when_value THEN statement_list]
...
[ELSE statement_list]
END as col_name;

循环语句

While
[begin_label_name:] WHILE search_condition DO
statement_list
END WHILE [begin_label_name];
  • 提前终止 while循环，则需要使用标签；标签需要成对出现。
  • 退出整个循环 leave(break)
  • 退出当前循环 iterate(continue)
  • 通过退出的标签决定退出哪个循环

窗口函数

也叫OLAP函数（Online Anallytical Processing，联机分析处理），可以对数据库数据进行实时分析处理

  • DENSE_RANK()：DENSE_RANK() OVER ([PARTITION BY col_name] ORDER BY col_name 排序规则)，生成连续序号，出现相同排名时，不跳过相同排名号，例：1，2，2，3
  • NTILE()：NTILE() OVER ([PARTITION BY col_name] ORDER BY col_name 排序规则)，按序号分组
  • RANK()：RANK() OVER ([PARTITION BY col_name] ORDER BY col_name 排序规则)，如果有并列名次的行，会占用下一名次的位置，但相同名词一致，例：1，2，2，4
  • ROW_NUMBER()：ROW_NUMBER() OVER ([PARTITION BY col_name] ORDER BY col_name 排序规则)，不考虑并列名次的情况，顺序排列，例：1，2，3，4
    聚合函数可以作为窗口函数，用法类似，在聚合函数参数位置填入需要计算的列名
  • lag() over()
  • lead() over()

其他函数

  • **LOAD_FILE(file_name)**：从文件读取内容
  • **ifnull(value_1,value_2)**：如果数据为null，将数据设为value_2；如果数据不为null，将数据设为value_1
  • cast(values as type)：将value转换为指定的类型type
  • convert(values,type)：将value转换为指定的类型type
  • 整数 : SIGNED
  • 无符号整数 : UNSIGNED
  • 浮点数 : DECIMAL
  • 字符型 : CHAR()，可带参数
  • 日期 : DATE
  • 时间 : TIME
  • 日期时间型 : DATETIME
  • 二进制，同带binary前缀的效果 : BINARY
  • Md5()：获取md5的hash值
  • Default()：指定默认值

变量

变量：允许用户自己定义变量，即为用户变量（用户自定义变量）

  • 定义变量
  • Set @var = value;
  • Select @v1:=id, @v2=name from tb_name limit 1：使用select into语句为变量初始化并赋值。要求select语句只能返回一行多个字段，就意味着同时为多个变量进行赋值，变量的数量需要与查询的列数一致。
  • select - from tb_name where @var:=30：通过select执行表达式完成变量初始化并赋值。
  • 为了避免=被当作关系运算符看待，使用赋值运算符:=代替。（set语句可以使用= 和 :=）。
  • 避免用户自定义的变量与系统标识符（通常是字段名）重复，用户自定义变量在变量名前使用@作为开始符号。
  • 变量被定义后，在整个会话周期都有效（登录到退出），会话结束（连接结束），释放变量，全局可用
  • 使用declare定义函数内变量，参数位置局部变量
  • Declare I int default 0;
  • 局部变量声明：
  • DECLARE var_name[,…] type [DEFAULT value]
  • 需要给变量提供一个默认值（DEFAULT子句），值可以被指定为一个表达式。如果没有DEFAULT子句，初始值为NULL。
  • 使用：语序使用 set 和 select into语句为变量赋值。
  • 注意在函数内是可以使用全局变量（用户自定义的变量的）

创建

  • 创建函数：

Create function function_name (参数列表：paramter_list) returns 返回值类型(返回值长度)
函数体：SQL_list

  • function_name：函数名，唯一标识函数，不能与已有的关键字冲突，可以使用db_name.funciton_name指明当前函数的所属数据库，否则为当前数据库。
  • paramter_list：参数部分，由参数名和参数类型组成。
  • 返回值类类型：一定要有return返回值
  • 函数体：由多条可用的mysql语句组成，包括流程控制、变量声明等语句构成。多条语句应该使用begin end语句块包含。
  • 删除函数：Drop function if exists function_name;
  • 查看函数：

Show function status like ‘partten’
Show create function function_name;

  • 修改函数：Alter function function_name 函数选项。
  • 调用函数：select function_name(paramter_lis);
例：随机生成学生名字
drop function if exists sname;
delimiter $$
create function sname () returns char(2)
begin
declare first_name char(16) default '赵钱孙李周吴郑王冯陈褚卫蒋沈韩杨';
declare last_name char(10) default '甲乙丙丁戊己庚辛壬癸';
declare full_name char(2);
set full_name = concat(substring(first_name, floor(rand()- 16+1), 1), substring(last_name, floor(rand()- 10+1), 1));
return full_name;
end
$$
delimiter ;
